# Oblique Collision of a Soft Rubber Disk with a Rigid Surface: Derivation and Explanation

This repository contains a comprehensive physics group project report that provides an explanatory guide and step-by-step mathematical derivation for the paper **"Oblique collision of a soft rubber disk with a rigid surface"**, originally published in the *American Journal of Physics* (AJP).

The project focuses on analyzing the complex mechanics behind the oblique (angled) impact of a deformable soft rubber disk against a solid, non-yielding wall, verified through video analysis and experimental tracking.

## Project Overview

When a soft rubber disk collides obliquely with a rigid surface, it undergoes significant deformation that fundamentally alters its post-collision trajectory, spin, and energy distribution compared to a perfectly rigid body. 

This project aims to:
- **Demystify the Mathematical Model:** Provide an accessible, step-by-step derivation of the equations governing the normal and tangential forces during impact.
- **Experimental Verification:** Replicate and analyze the collision scenarios using video analysis software (**Tracker**) to compare empirical results with theoretical models.
- **LaTeX Implementation:** Compile the entire analytical breakdown into a publication-quality PDF report using LaTeX.
